# Form Fields

## Purpose

This file documents the main fields used by the SAP Build Process Automation form.

## Complaint Fields

- Customer name
- Complaint text
- Complaint source
- Submitted date

## AI Triage Fields

- Summary
- Category
- Priority
- SLA risk
- Recommended team
- Workflow route
- Confidence score
- Requires human review
- Decision justification

## Notes

These fields are based on the structured JSON output created during SAP Generative AI Hub testing. In Phase 1, the values are manually copied into SAP Build for workflow testing.
