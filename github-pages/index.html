SAP BTP Business AI Complaint Triage

A completed Phase 1 MVP showing AI-assisted complaint triage, SAP Build workflow routing, and S/4HANA process mapping.

What it demonstrates

  • SAP Generative AI Hub prompt testing and structured JSON output.
  • SAP Build Process Automation workflow and decision-table routing.
  • SAP S/4HANA process mapping for finance, sales, service, and supply chain context.

MVP boundary

AI output is manually transferred into SAP Build in Phase 1. Live API and OData integrations are planned future enhancements.
